# Zelda Wiki

## Uma Wikipedia Zelda

Minha missão é criar uma aplicação web que esteja conectada a uma api de dados sobre a franquia de jogos The Legend Of Zelda. Os objetivos incluem:

> Aprimorar habilidades na biblioteca React <br>
> Aprimorar habilidades com Typescript
<br>
> Utilizar uma api publica
<br>
> Fazer um projetinho da minha franquia de jogos favorita.

## Recursos usados:

Api externa: https://docs.zelda.fanapis.com/
