Minha missão aqui é criar uma aplicação web que esteja conectada a uma api de dados sobre a franquia de jogos The Legend Of Zelda. Os objetivos incluem:

> Aprimorar habilidades na biblioteca React <br>
> Aprimorar habilidades com Typescript <br>
> Utilizar uma api publica <br>
> Fazer um projetinho da minha franquia de jogos favorita.

## Sobre Zelda:
The Legend of Zelda é uma série de jogos eletrônicos de ação-aventura com alguns elementos de RPG, criada pela Nintendo em 1986 por Shigeru Miyamoto e Takashi Tezuka exclusivamente para os consoles da Nintendo.

## Recursos usados:

Api externa: https://docs.zelda.fanapis.com/

Github da Api: https://github.com/deliton/zelda-api
